## Key Notes

> **Virtual key pattern** — `SMARTROUTER_KEY_{SLUG}` / `SMARTROUTER_ANTHROPIC_KEY_{SLUG}` / `SMARTROUTER_BUDGET_{SLUG}` in Vercel env. Real `sk-ant-*` keys live only in Vercel (masked). Client apps hold worthless `sk-sr-*` keys. Add new clients via Doppler + Vercel env vars only — do not put real keys in Render.

> **Breach baseline** — `BASELINE_START_DATE` is set to exclude the Apr 20–25 attacker usage from anomaly baselines. Do not remove this until 30+ days of clean data accumulate.

> **`proxy.ts` vs `middleware.ts`** — The Clerk middleware file is named `proxy.ts` (renamed at `567ec1b` for Next.js 16 compatibility). This is intentional. Do not rename back to `middleware.ts`.
